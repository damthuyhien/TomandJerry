Sinh viên làm việc nhóm 3 người

Bước 1: Tom - Thiết lập dự án

  • Khởi tạo một kho Git mới
  • Sử dụng lại mã nguồn trang web này, lưu trữ dưới tên colab_git.html
  • Tạo một nút ấn Press me!
  • Đẩy kho lưu trữ lên GitHub (nhánh chính: main)

Bước 2: Jerry - Thêm giao diện

  • Clone kho lưu trữ
  • Tạo một nhánh mới feature_styles
  • Chỉnh sửa styles.css để trang web này đẹp nhất có thể và commit thay đổi
  • Đẩy nhánh lên và tạo pull request vào main

Bước 3: Spikee - Thêm tương tác

  • Clone kho lưu trữ
  • Tạo một nhánh mới feature_scripts
  • Chỉnh sửa script.js để thêm chức năng JavaScript sau khi ấn Press me thì hiển thị tên 3 thành viên nhóm
  • Đẩy nhánh lên và tạo pull request vào main
